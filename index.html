Root Element (<html>):

  • The <html> element represents the top-level element of an HTML document.
  • It defines the entire HTML document structure.
  • All other elements must be descendants of the <html> element.
  • The <html> element contains the <head> and <body> elements.

Meta Data Elements (<meta>):

  • The <meta> element represents meta-information about the HTML document.
  • It is typically used to specify the character set, page description, keywords, author of the document, and viewport settings.
  • Metadata will not be displayed on the page, but is machine parsable.
  • It is used by browsers (how to display content or reload page), search engines (keywords), and other web services.
  • <meta> tags always go inside the <head> element, and are typically used to specify character set, viewport settings, and other metadata about the HTML document.

Headings (<h1> to <h6>):

  • These elements represent six levels of section headings in HTML.
  • <h1> is the highest section level and <h6> is the lowest.
  • They help in defining the structure of the document and also help with SEO.

Paragraph (<p>):

  • The <p> element represents a paragraph in HTML.
  • It is a block-level element and is used to group sentences and parts of text together.

Division (<div>):

  • The <div> element is a generic container for flow content, which does not inherently represent anything.
  • It can be used to group elements for styling purposes, or because they share attribute values.

Span (<span>):

  • The <span> element is a generic inline container for phrasing content, which does not inherently represent anything.
  • It can be used to group elements for styling purposes, or because they share attribute values.

Blockquote (<blockquote>):

  • The <blockquote> element represents content that is quoted from another source.
  • Content inside a <blockquote> must be quoted from another source, whose address, if it has one, may be cited in the cite attribute.
